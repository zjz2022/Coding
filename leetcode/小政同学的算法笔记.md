 二、连续子数组最大和(42) 

动态规划

1、先存下第一个作为最大值

2、然后判断前一位是否大于零，大于零就令

num[i]+=num[i-1]

相当于是吧num[i]位合并成一个数组了

3、将max和nums[i]位判断，相当于是把max值存下来

4、返回max

 三、数组中出现次数超过一半的数字(39) 

解法1:

开辟一个额外空间存储每个值出现的次数，时间复杂度最大为O(n)，逻辑简单



用哈希表来存储

1、第一次存的时候map.set(nums[i],i)

2、如果再次遇到就map.set(nums[i],map.get(nums[i])+1)

 五、把数组排列成最小数(45)

如果return值<0,就会从小到大排

如果return值>0,就会从大到小排



所以

let one=''+a+b

let two=''+b+a

return one-two
